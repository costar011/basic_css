css의 정의 방법 Inline Style Sheet (인라인 스타일시트 형식) Embedded Style Sheet (임베디드 스타일시트 형식) External Style Sheet (외부 스타일시트 형식) Imported Style Sheet (임포티드 스타일시트 형식)
css 속성 여러 속성 값은 반드시 공백으로 구분되어야 한다. 축약 표현 속성은 여러 속성 값을 하나의 간소화된 선언으로 적용할 수 있다. 속성이 명시되지 않으면 해당 속성의 기본 값이 적용된다.
